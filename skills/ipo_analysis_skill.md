# Skill: IPO Analysis

*Layer: L1 — Core Skills*

## Purpose
Analyze an IPO using Akshat's framework: business quality, valuation discipline, promoter credibility, and use of proceeds.

## Execution Steps

### Step 1: Business Model Quality
- What does the company do?
- Is it in a profitable or expanding sector?
- Does it have moat characteristics (pricing power, switching costs, network effects)?
- Is it pre-revenue, early-profit, or mature?

### Step 2: Valuation at IPO Price
- PE at upper price band
- PE vs best listed peer
- If PE > 2x listed peers with no clear moat → SKIP
- Grey Market Premium (GMP): use as sentiment signal ONLY, not fundamental anchor

### Step 3: Use of Proceeds
- Debt repayment: ⚠️ RED FLAG (promoters exiting debt, not building business)
- OFS (Offer for Sale): ⚠️ RED FLAG (promoters selling, not issuing new shares)
- Capex / growth investment: NEUTRAL to POSITIVE
- Working capital: NEUTRAL

### Step 4: Promoter Assessment
- Track record in other businesses?
- Prior capital allocation history?
- Is this their first listed entity or part of a proven group?

### Step 5: Listing Strategy
- Short-term listing gain vs long-term holding?
- If listing gain strategy: cap allocation at 2–5%
- If long-term strategy: treat like any stock purchase, apply full stock analysis

## Output Format
1. Verdict: Apply / Skip / Wait for Listing
2. Business Quality: Strong / Average / Weak
3. Valuation vs Peers: Cheap / Fair / Expensive / Avoid
4. Proceeds Flag: Clean / Warning
5. Promoter Assessment
6. Recommended Allocation (% of portfolio, if applying)
