# Performance Reports

This directory (`performance/reports/`) is designated for storing future accuracy reports.

As closed predictions accumulate in the `predictions/closed/` directory, human reviewers or future knowledge compilation runs will generate periodic reports analyzing the system's prediction accuracy.

## Report Structure

Future reports stored here should follow standard naming conventions, such as `YYYY-MM-DD_Q3_Performance_Report.md`.

Reports should summarize the metrics defined in `performance/scoring_framework.md`:
- Overall Accuracy
- Sector / Theme Accuracy
- Confidence Calibration analysis
- Qualitative Error Analysis and Lessons Learned
