# Test: Macro Analysis Workflow (WF-03)

## Test Case 1 — Liquidity Tightening Environment

**Input:** "Is now a good time to deploy capital in India?"
**Context:** Rate hikes ongoing, FII outflows, INR weakening, earnings flat

**Expected Output:**
- [ ] Liquidity: Tightening
- [ ] Rate Direction: Hiking
- [ ] India Risk: High
- [ ] Drawdown Classification: Potential structural pressure
- [ ] Recommended Posture: Defensive or Neutral
- [ ] INR depreciation risk explicitly stated
- [ ] US/global alternative mentioned as better near-term option
