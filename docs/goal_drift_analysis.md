# Goal Drift Analysis

## Overview
The original goal was: **Find emerging trends early → Convert to sectors → Convert to stocks → Reject bad ideas → Produce watchlists.**
Over the course of the architectural phases, the project suffered severe goal drift, morphing from an **Opportunity Discovery System** into a **Static Knowledge Management Archive**.

---

## Areas of Critical Goal Drift (Ranked by Severity)

### 1. The Ontology Map (Taxonomy Building vs. Trend Detection)
* **The Drift:** Instead of building an engine that reads the news and infers a trend, the project built `ontology_map.json`—a hardcoded dictionary mapping "rate cut" to `liquidity_up`.
* **Why it's fatal:** This is the opposite of discovery. You cannot discover a trend if you must manually hardcode the exact keyword into a dictionary before the system can recognize it.

### 2. The Promotion Thresholds (Archiving vs. Opportunity)
* **The Drift:** The update engine requires a principle to be stated 8+ times over 2+ time periods before becoming "Doctrine" in the Master System.
* **Why it's fatal:** By the time Akshat has stated an idea 8 times across 2 years, the trend is already market consensus. The asset has already repriced. This mechanism builds a perfect historical archive but destroys "early" discovery.

### 3. The Flat Collision Engine (Routing vs. Generation)
* **The Drift:** The MVP collision engine (`collide.py`) does not generate a framework. It routes an input string to an output string via `activation_tags`. It is an `if-then` routing script.
* **Why it's fatal:** It never actually connects the trend to a sector, nor a sector to a stock. It stops at "Here is a principle." It entirely abandoned the bottom half of the original goal (Sector -> Stock -> Watchlist).

### 4. Over-indexing on Contradiction Logs (Documentation vs. Alpha)
* **The Drift:** Massive architectural focus was placed on Doubt Engines, Contradiction Logs, and Human-in-the-loop PR gating to protect the sanctity of `01_Akshat_Master_System.md`.
* **Why it's fatal:** While intellectually rigorous, it turned the project into an academic wiki. No alpha is generated by proving Akshat contradicted himself in 2024. Engineering hours were spent on documentation integrity rather than market screening.

### 5. Stripping the NLP/Embeddings for the MVP
* **The Drift:** In an attempt to make the MVP "simple" and "deterministic," all semantic search, LLMs, and embeddings were banned.
* **Why it's fatal:** The original goal *requires* semantic inference. You cannot convert an abstract macro trend ("Demographic collapse in China") into a specific stock watchlist using deterministic regex and JSON routing. The constraint destroyed the capability.
