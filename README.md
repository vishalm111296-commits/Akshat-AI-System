# Akshat-AI-System

> **A self-improving investing intelligence system built on Akshat Shrivastava's publicly available frameworks, videos, community posts, and research material.**

---

## What This Is

This repository is a **production-grade, modular, evidence-driven knowledge system** designed to replicate and continuously update Akshat Shrivastava's investing philosophy. It is not a content archive. It is an operating intelligence system.

## Architecture (10 Layers)

| Layer | Directory | Responsibility |
|---|---|---|
| L0 | `operating_system/` | Ground rules, glossary, repo constitution |
| L1 | `skills/` | Reusable analytical modules |
| L2 | `knowledge/` | Permanent doctrine (human-gated) |
| L3 | `knowledge/` | Dynamic tactical memory (auto-updatable) |
| L4 | `raw_sources/` | Verbatim evidence — never modified |
| L5 | `routing/` | Knowledge and skill routing logic |
| L6 | `workflows/` | End-to-end workflow definitions |
| L7 | `update_engine/` | Promotion and update rules |
| L8 | `automation/` + `.github/workflows/` | Free automation (GitHub Actions + Python) |
| L9 | `docs/` + `scripts/` | AI and human query interface |

## Quick Start

1. Read `operating_system/system_rules.md` first
2. For analysis tasks → start at `routing/05_Akshat_Knowledge_Router.md`
3. For adding new sources → follow `docs/how_to_add_source.md`
4. For understanding doctrine → read `knowledge/01_Akshat_Master_System.md`

## Critical Rules

- `01_Akshat_Master_System.md` is **NEVER modified by automation**
- Raw sources are **NEVER deleted** — only archived
- AI-generated content is **NEVER counted as a source**
- One source = one count maximum in the Frequency Table
