## Critical Rules

- `01_Akshat_Master_System.md` is **HUMAN-ONLY**. Automation never modifies it.
- One mention never updates doctrine. Minimum **3 independent sources** required for promotion.
- Never count generated documents in frequency scores.
- Raw Sources are append-only. Never modify after ingestion.
